# Kontakts
 Spēle brīvā dabā! 
 
 Device that detects other players if they are within reach. \
 Cypress CYBLE-416045-02 Bluetooth 5.0 Module, WS2812B individually addressable LEDs, vibro-motors, magnets and 3D printed enclosure!
